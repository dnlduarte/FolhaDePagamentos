# 💰 Calculadora de Folha de Pagamento em Java

Projeto desenvolvido em **Java** para calcular o **salário líquido** de um funcionário com base nas informações fornecidas pelo usuário.

A aplicação utiliza **JOptionPane** para capturar dados e exibir o resultado em janelas de diálogo.

---

## 🚀 Funcionalidades

O programa solicita ao usuário:

* Salário bruto
* Número de dependentes
* Percentual de desconto do INSS
* Valor do plano de saúde

Após inserir os dados, o sistema calcula e exibe:

* Salário bruto
* Número de dependentes
* Desconto do INSS
* Valor do plano de saúde
* **Salário líquido**

🖥️ Exemplo de execução

O programa abre caixas de diálogo solicitando as informações do usuário e depois apresenta o resultado final com todos os dados formatados.

--saída:

Salário Bruto: R$ 3000.00
Dependentes: 2
INSS: 8%
Plano de Saúde: R$ 200.00
Salário Líquido: R$ 2560.00
